# pinterest-export

CLI tool to scrape Pinterest boards, AI-analyze pins, and generate HTML galleries with context files.

## Features (planned)
- Scrape any Pinterest board (public)
- AI vision analysis of each pin (descriptions, tags, themes)
- Generate beautiful HTML gallery with metadata
- Export context files for downstream use (design research, mood boards, etc.)

## Install
```
pip install pinterest-export
```

## Usage
```bash
pinterest-export https://pinterest.com/username/boardname --output ./gallery
```
